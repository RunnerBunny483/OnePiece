USER GUIDE FOR GENERATING ONE PIECE WANTED POSTERS


Welcome to the One Piece Wanted Poster generator's guide.


This program will allow you to generate wanted posters and reports based on the famous animated serie called One Piece, it has three uses: Create Poster, Select Poster and Generate Report.

Main Menu

When you open this application you will see many buttons:

  • Create Poster: It allows you to generate a custom wanted poster using any name, bounty and image.
  • Select Poster: It allows you to search for a wanted poster about a pirate using its name.
  • Generate Report: It allows to generate a report about pirates using its bounties.

We can change the language if we want by using the pull-down menu related to the languages.

Create Poster

In this section you can custom:

  • The name.
  • The bounty.
  • The picture.
  • And press the button Create Poster to create a custom wanted poster.

Select Poster

In this section you can search for:

  • The name.
  • And press the button Select Poster to search a pirate and generate its wanted poster.

Generate Report

In this section you can search for:

  • A certain bounty up the one specified.
  • The crew of the pirate.
  • And press the button Generate Report to generate the report.

Report Generation

The wanted posters and reports will be generating in PDF format using JasperReports and will be storaged in the folder of the project relating JasperReports.
