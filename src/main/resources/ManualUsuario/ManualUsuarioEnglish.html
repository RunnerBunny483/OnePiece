USER GUIDE FOR GENERATING ONE PIECE WANTED POSTERS


Welcome to the One Piece Wanted Poster generator's guide.


This program will allow you to generate wanted posters and reports based on the famous animated serie called One Piece, it has three uses: Create Poster, Select Poster and Generate Report.

Main Menu

When you open this application you will see many buttons:

  • Create Poster: Permite generar un cartel personalizado con un nombre, recompensa y la imagen que se quiera.
  • Select Poster: Permite generar un cartel sobre un pirata según su nombre.
  • Generate Report: Permite generar un informe sobre piratas según sus recompensas.

Podremos cambiar el idioma de la aplicación eligiéndolo en la menú desplegable de Idioma.

Create Poster

En esta sección, puedes personalizar:

  • El nombre.
  • La recompensa.
  • La imagen.
  • Y presionar el botón Crear Póster para crear tu cartel personalizado.

Select Poster

En esta sección, puedes buscar:

  • El nombre.
  • Y presionar el botón Buscar Pirata para buscar un pirata y generar su cartel correspondiente.

Generate Report

En esta sección, puedes seleccionar:

  • Una recompensa específica de hasta el valor introducido.
  • La tripulación del personaje.
  • Y presionar el botón Generar Informe para generar el informe.

Generation of Reports and Wanted Posters

The wanted posters and reports will be generating in PDF format using JasperReports and will be storaged in the folder of the project relating JasperReports.
